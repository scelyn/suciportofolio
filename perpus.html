Library Management System

Sistem Informasi Manajemen Perpustakaan

Latar Belakang

Sistem ini dikembangkan untuk mendigitalisasi proses peminjaman buku yang sebelumnya dilakukan secara manual. Fokus utama proyek ini adalah integritas data dan kemudahan pencarian koleksi bagi mahasiswa.


Fitur Implementasi

  • Dashboard Admin: Mengelola inventaris buku dan denda.
  • Sistem Peminjaman: Validasi otomatis ketersediaan buku.
  • Responsive UI: Dapat diakses melalui smartphone.

Teknologi

PHP Laravel, MySQL Database, Bootstrap 5, dan VS Code sebagai editor utama.
